# Freelance Flow

A freelance workflow management application built with React, TypeScript, Vite, and Supabase.

## Current Progress
- Initialized React project with Vite (React + TypeScript).
- Setup Supabase connection configuration (`credentials.js` / `.env`).
- Basic project structure setup in `src/` and `public/` directories.

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
